# 宠物社区小程序首页 UI 设计说明

## 1. 设计概述

### 项目背景
为用户 Ren 设计一款宠物社区类小程序，首页风格参考小红书的卡片式瀑布流布局，打造温馨、友好的宠物内容分享社区。

### 设计目标
- 打造温暖、活泼的视觉风格，符合宠物社区的温馨感
- 借鉴小红书的瀑布流布局，呈现丰富的内容卡片
- 提供流畅的移动端交互体验

---

## 2. 配色方案

### 主色调
| 颜色名称 | 色值 | 用途 |
|---------|------|------|
| Primary | `#FF6B6B` | 主色调（珊瑚红），用于品牌标识、点赞激活态 |
| Secondary | `#4ECDC4` | 辅助色（薄荷绿），用于分类标签 |
| Accent | `#FFE66D` | 强调色（暖黄），用于特殊标签 |

### 宠物主题色板
| 颜色名称 | 色值 | 用途 |
|---------|------|------|
| Cute Blue | `#6B9FFF` | 猫咪分类、喵星人标签 |
| Cute Pink | `#FF8EC4` | 狗狗分类、汪星人标签 |
| Cute Purple | `#B388FF` | 兔兔、鸟类分类标签 |
| Cute Green | `#7FD8A6` | 日常分享标签 |
| Cute Orange | `#FFB366` | 仓鼠分类标签 |

### 暖色系（背景/辅助）
| 颜色名称 | 色值 | 用途 |
|---------|------|------|
| Warm 50 | `#FFF9F5` | 页面背景（微暖白） |
| Warm 100 | `#FFF0E8` | 卡片悬停背景 |
| Warm 600 | `#FF6B6B` | 主色调同系 |

### 设计原则
- **温馨感**：使用珊瑚红 `#FF6B6B` 作为主色，传递温暖与活力
- **友好感**：丰富的柔和色调，让用户感到放松和愉悦
- **辨识度**：不同宠物类型使用专属颜色，便于快速识别

## 3. 字体选择

### 主字体
- **字体名称**：Noto Sans SC（思源黑体）
- **字重**：400（正文）、500（中等）、600（标题）、700（强调）
- **用途**：全部文本内容
- **特点**：Google Fonts 中文字体，清晰易读，完美支持中文

### 设计原理
根据 UI/UX 设计研究，宠物社区适合使用：
- **Playful Creative** 风格：友好、活泼、现代
- **圆润友好** 的视觉感受，符合宠物主题

## 4. 核心组件说明

### 4.1 顶部导航栏（Header）

**结构**：
```
┌────────────────────────────────────────┐
│ 🐾 Logo │ 搜索框(可点击) │ 🔔 消息     │
├────────────────────────────────────────┤
│ 推荐 | 🐶狗狗 | 🐱猫咪 | 🐰兔兔 | ...  │
└────────────────────────────────────────┘
```

**设计要点**：
- Logo 使用渐变背景（Primary → Cute Pink）的圆角方形
- 搜索框圆角胶囊形状，背景灰色 `#F3F4F6`
- 消息图标带红点提示，增强互动感
- 分类标签横向滚动，支持拖拽操作

### 4.2 瀑布流内容卡片（Masonry Card）

**卡片结构**：
```
┌──────────────────┐
│  [图片区域]      │
│  🐶 汪星人       │ ← 分类标签（半透明背景）
├──────────────────┤
│ 标题文案...      │
├──────────────────┤
│ 👤 用户名  ❤️123 │
└──────────────────┘
```

**设计要点**：
- **双列瀑布流**：column-count: 2，column-gap: 12px
- **圆角**：`rounded-2xl`（20px），营造柔和感
- **阴影**：`shadow-sm`，轻微投影增加层次
- **悬停效果**：上移 2px + 阴影加深
- **点击反馈**：scale(0.98) 按压效果

### 4.3 分类标签（Category Tag）

**设计特点**：
- 圆角胶囊：`rounded-full`
- 半透明背景 + 毛玻璃效果（backdrop-blur-sm）
- 不同宠物类型使用专属 emoji
- 激活态：实心背景 + 白色文字

### 4.4 底部 TabBar

**结构**：
```
┌────────────────────────────────────────┐
│  🏠   🔍   ➕   💬   👤                │
│ 首页  发现  发布  消息  我的            │
└────────────────────────────────────────┘
              ↑
         凸起的发布按钮
```

**设计要点**：
- 5个 Tab 项均分宽度
- **首页** 激活态：Primary 色 + 实心图标
- **发布按钮**：圆形渐变背景，向上偏移 `-mt-4`，视觉突出
- 消息 Tab 带红点未读提示
- 支持 iOS 安全区域（safe-area-inset-bottom）

### 4.5 交互元素

#### 点赞按钮
- **未点赞态**：灰色 `#9CA3AF` + 空心心形
- **点赞态**：Primary `#FF6B6B` + 实心心形
- **动画**：点击时心形放大缩小的弹跳效果（heartBeat）
- **数字更新**：支持 k/w 缩写（如 2.3k, 5.8k）

#### 发布按钮
- 渐变背景：`from-primary to-cute-pink`
- 投影：`shadow-lg`
- 点击缩放：`active:scale-95`

## 5. 技术实现

### 技术栈
- **HTML5**：语义化结构
- **Tailwind CSS**：通过 CDN 引入，响应式工具类
- **Vanilla JavaScript**：原生交互逻辑
- **Google Fonts**：Noto Sans SC 字体

### 响应式设计
- **视口配置**：`maximum-scale=1.0, user-scalable=no` 禁止缩放
- **移动优先**：基础宽度 375px（标准移动端）
- **安全区域**：支持 iPhone 刘海屏（env(safe-area-inset-top)）

### 性能优化
- **图片懒加载**：`loading="lazy"`
- **GPU 加速**：动画使用 transform/opacity
- **硬件优化**：移除滚动条自定义样式

## 7. 关键设计决策

### 为什么选择这些设计？

| 决策点 | 选择 | 原因 |
|-------|------|------|
| **配色** | 珊瑚红 + 暖色调 | 宠物社区需要温馨感，红色传递热情与关爱 |
| **布局** | 瀑布流双列 | 参考小红书，符合内容型社区的用户习惯 |
| **圆角** | 16-24px 大圆角 | 柔和友好的视觉感受，降低攻击性 |
| **卡片** | 圆角 + 轻阴影 | 层次分明但不厚重，符合轻量化趋势 |
| **字体** | 思源黑体 | 中文支持优秀，清晰可读 |
