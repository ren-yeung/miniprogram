## 如何打开项目

### 方法1：使用微信开发者工具GUI
1. 打开微信开发者工具
2. 点击「导入项目」
3. 选择项目目录：`/app/data/所有对话/主对话/宠物社区`
4. AppID会自动识别：`wxc125919431b74f14`
5. 点击「确定」

### 方法2：使用命令行CLI
1. 首先需要在微信开发者工具中开启CLI服务端口：
   - 打开微信开发者工具
   - 点击「设置」→「安全设置」
   - 勾选「开启服务端口」
2. 然后在终端中运行：
   ```bash
   wechat-devtools-cli open --project /app/data/所有对话/主对话/宠物社区
   ```

## 功能概览

### TabBar页面（4个）
1. **首页** - Banner轮播、功能图标区、瀑布流笔记
2. **广场** - 分类Tab、悬浮发布按钮、帖子列表
3. **平台** - 核心工具、本地服务、科普知识
4. **我的** - 用户卡片、数据统计、功能菜单

### 额外页面
- **帖子详情** - 查看完整帖子内容和评论
- **搜索页** - 搜索笔记、商品、用户
- **发布页** - 发布晒宠、送养、求助等

## 设计规范

- **主色调**：#FF6B6B 珊瑚红
- **背景色**：#FFF9F5 奶油白
- **圆角**：12-24rpx
- **字体**：圆润无衬线
- **风格**：小红书式轻社区 + 软萌治愈风

## 注意事项

1. TabBar图标为占位符图片，实际使用时需替换为正式图标
2. 所有网络图片使用picsum.photos和i.pravatar.cc作为占位图
3. 项目已配置分包策略优化加载速度
